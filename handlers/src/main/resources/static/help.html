Как посмотреть список продуктов?

Для просмотра списка продуктов есть GET-запрос /products/list.
Он выдает HTML-документ с таблицей значений и формой отправки для создания продуктов.

Как добавить продукт?

Для добавления продукта есть POST-запрос /products/add.
Необходимо передать x-www-form-urlencoded с 3 параметрами:
* name: название продукта, не пустая строка
* manufacturer: имя производителя, не пустая строка
* amount: количество товара, не отрицательное число
Также можно использовать форму для отправки в списке продуктов.

Какие данные авторизации?

Существует два пользователя:
* iamguest: пароль iamguest, роль guest
* iammanager: пароль iammanager, роль manager
Менеджер может добавлять продукты и посмотреть их список.
Гость имеет право только посмотреть список продуктов.
